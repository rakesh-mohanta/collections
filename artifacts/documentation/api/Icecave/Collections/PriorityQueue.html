Icecave\Collections\PriorityQueue

class PriorityQueue extends Queue implements Serializable

A prioritized first-in/first-out (FIFO) queue of elements.

Higher priority values are moved closer to the front of the queue.

Prioritization is provided by the prioritzation function specified in the constructor, but may be optionally overridden by the second parameter to {see PriorityQueue::push()}.
